Mendeley adalah aplikasi *reference manager* gratis.

Fitur utama:
- Import sitasi dari browser langsung ke library.
- Generate daftar pustaka otomatis di Microsoft Word.
- Membaca dan memberi anotasi langsung pada file PDF.
